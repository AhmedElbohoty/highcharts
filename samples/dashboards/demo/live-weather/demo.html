Live weather demo

The demo shows weather forecasts from the Norwegian Meteorological Institute in four Dashboard components.

Dashboard components

  • Highcharts Maps markers and tooltip show the selected parameter.
  • Dashboard KPI shows station info, temperature, pressure and humidity of the selected station.
  • Dashboard Datagrid shows all the parameters of the selected station for the next 24 hours.
  • Highcharts Spline Chart shows the selected parameter of the selected station for the next 24 hours.

Data set

The data shown in this demo are weather forecasts containing temperature, humidity and pressure from a selection of weather stations around the world.

  • The station is selected by clicking on the map
  • The weather parameter (e.g. humidity) clicking on an item in the KPI component.

Customizing the demo

The demo offers limited customization possibilities directly in the code:

  • New weather stations can be added (and existing deleted).
  • Alternative sets of weather stations can be used.
  • The range of the data set is configurable. By default the sample start from the latest observation and 24 hours into the future. Note that the forecast covers 19 days.
